## Getting Started

1. Configure settings in the CableKnit dashboard
2. Add automations in the `automations/` directory
3. Add skills in the `skills/` directory
4. Add artifact blueprints in the `blueprints/` directory
5. Add data source tools in the `tools/` directory
6. Run `cableknit validate` to check your bundle
7. Run `cableknit push` to publish
